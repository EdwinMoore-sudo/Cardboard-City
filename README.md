# Cardboard City - The Interactive City Builder

## An interactive simulation for designing and maintaining a prosperous community while managing local resources.

This project was created during my senior year of high school as part of a semester-long software development project. The goal was to design, 
prototype, and develop a fully interactive city-building simulation within a few months while applying object-oriented programming principles 
and software engineering practices.

Players manage a growing settlement by constructing buildings, assigning citizens to jobs, scheduling daily tasks, and balancing available resources. 
As the in-game clock advances, citizens perform scheduled activities such as building, clearing land, gathering resources, farming, hunting, fishing, 
and resting. These interconnected systems work together to simulate the growth and management of a small community.

## Features
  - Tile-based city-building gameplay
  - Interactive construction and building placement
  - Citizen management and workforce assignment
  - Hourly task scheduling system
  - Real-time simulation clock with day progression
  - Drag-and-drop task assignment interface
  - Custom-built menu and UI system

## Technical Highlights

  - Object-Oriented Programming
  - Event-Driven Programming
  - Data Structures & Algorithms
